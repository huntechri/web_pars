# Frontend (Next.js)

Минимальный UI:
- авторизация
- выбор категорий (включая дочерние)
- запуск парсинга
- скачивание CSV

## Запуск

1. Скопируйте `.env.local.example` в `.env.local`
2. Установите зависимости: `npm install`
3. Запустите: `npm run dev`

По умолчанию фронт доступен на `http://localhost:3000`.
