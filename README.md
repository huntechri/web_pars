# 🛠️ Парсер Петрович Pro

Профессиональный инструмент для автоматического сбора данных о товарах с сайта Петрович с поддержкой 4 уровней категорий, параллельной загрузки и удобным GUI.

---

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/Nastya870/parser_petr_full.git
cd parser_petr_full
```

### 2. Создайте виртуальное окружение и установите зависимости

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Сгенерируйте дерево категорий (первый раз, ~5-10 минут)

```powershell
python build_full_categories_tree.py
```

### 4. Запустите приложение

```powershell
python app_gui.py
```

## 📋 Возможности

- ✅ **Глубокий парсинг** — до 4 уровней категорий  
- ✅ **Параллельная загрузка** — 5 категорий одновременно (ускорение в 3-5 раз)  
- ✅ **Мгновенный запуск GUI** — категории загружаются из конфига за 1 секунду  
- ✅ **Автосохранение настроек** — выбранные категории и колонки сохраняются автоматически  
- ✅ **Настройка колонок** — выберите какие поля экспортировать в CSV  
- ✅ **Древовидная структура** — раскрывайте только нужные категории  
- ✅ **Управление файлами** — открывайте и удаляйте результаты прямо из GUI

## 🎯 Использование

### Шаг 1: Выбор категорий

1. В древовидной структуре раскройте нужные группы (▶)
2. Выберите категории (☑)
3. При выборе родительской категории автоматически парсятся **все** дочерние

### Шаг 2: Настройка колонок

Выберите колонки для экспорта:
- `Уровень 1-4` — иерархия категорий
- `Артикул` — код товара
- `Название` — наименование
- `Цена` — стоимость
- `Ед. изм.` — единица измерения
- `Бренд` — производитель
- `Вес` — масса товара
- `Изображение` — ссылка на фото
- `Ссылка` — URL товара

### Шаг 3: Запуск парсинга

1. Установите лимит (опционально)
2. Нажмите **«ЗАПУСТИТЬ ПАРСИНГ»**
3. Результат сохранится в CSV файл

## 📝 Решение проблем

### Ошибка 403 Forbidden

Куки устарели. Обновите их:
1. Откройте сайт Петрович в браузере
2. DevTools (F12) → Network → скопируйте куки
3. Обновите через GUI или файл `Cook`

### GUI не открывается

```powershell
python build_full_categories_tree.py
```

### Слишком долго парсится

- Установите лимит товаров
- Выбирайте конкретные категории
